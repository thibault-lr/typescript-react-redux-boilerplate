# React redux boilerplate 

Simple react boiler plate using :
- Typescript
- React 
- Redux
- Webpack 4
- Sass
- Jest 
- 

# Install 
```yarn install```

# Dev server 
```yarn start```

# Test
```yarn test```

# Build for production 
It will create a build folder 
```yarn build```
